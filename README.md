# NBA Wemby Climate Project

## Graph Description
The visualization pairs an interactive U.S. map with a multi-line season trend chart for NBA teams.
Users can select teams from the map or the multi-select list, then switch the chart metric between pace,
field goal attempts per game (FGA/G), three-point attempts per game (3PA/G), and wins.

Each selected team is drawn as its own colored line with point markers by season.
A dashed league-average line is included to show how each team compares to the overall NBA trend over time.
Hovering over chart points reveals the exact season and value for the selected metric.
