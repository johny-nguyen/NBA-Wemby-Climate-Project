# NBA Wemby Climate Project

## Graph Description
The visualization pairs an interactive U.S. map with a multi-line season trend chart for NBA teams.
Users can select teams from the map or the multi-select list, then switch the chart metric between pace,
field goal attempts per game (FGA/G), three-point attempts per game (3PA/G), and wins.

Each selected team is drawn as its own colored line with point markers by season.
A dashed league-average line is included to show how each team compares to the overall NBA trend over time.
Hovering over chart points reveals the exact season and value for the selected metric.

## Project Progress and Challenges
Our team’s final project aims to create an interactive data visualization experience that helps users better understand patterns and insights within our chosen dataset through engaging and intuitive visual storytelling. We plan to use D3.js to build dynamic and interactive components that allow users to explore the data from multiple perspectives, following the course requirement of publishing an interactive web visualization.

So far, we have gathered and cleaned our dataset, explored possible research questions, and begun designing the structure and layout of our visualizations. We have also started implementing the foundational components of the website, including initial charts, interaction ideas, and navigation flow.

One of the biggest challenges of this project will be balancing visual complexity with clarity. Since interactive visualizations can easily become overwhelming, we want to ensure that users can meaningfully explore the data without confusion. Another challenge will be implementing smooth and responsive D3.js interactions while maintaining good performance and accessibility across the webpage.
